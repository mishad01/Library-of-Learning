Lesson Aim
To extract data from the database, I  need to use the structured query language (SQL) SELECT
statement. I may need to restrict the columns that are displayed. This lesson describes all the
SQL statements that I need to perform these actions.

QUESTIONS 
1. The following SELECT statement executes successfully:
SELECT last_name, job_id, salary AS Sal
FROM employees;
True/False

2. The following SELECT statement executes successfully:
SELECT *
FROM job_grades;
True/False

3. There are four coding errors in this statement. Can you identify them?
SELECT employee_id, last_name
sal x 12 ANNUAL SALARY
FROM employees;
